# TechRecruit — recruitment website

Next.js site for **TechRecruit**, a technology recruitment agency: public job listings, candidate apply flow, Firebase-backed applications, and a recruiter portal. Configure environment variables from `env.example` (Firebase, optional Java backend for vacancies and screening merge).

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Stack

Next.js (App Router), React, Tailwind CSS, Firebase Auth & Firestore, optional server-to-server calls to your API for vacancies and tenant application screening.
